
# CLAUDE.md

## WHY: プロジェクトの目的と前提
株式会社メディカバイスのランディングページ開発プロジェクト。
投資を検討する人など、信用が必要な人に対し、事業や人の信用性を訴求し、お問い合わせへのコンバージョン率(CVR)を最大化する。
**対象者**: [開発者/デザイナー/AIエージェント]
**言語**: 日本語

## WHAT: 技術スタックと構成要素
**技術スタック**: [Next.js / React / Tailwind CSS / TypeScript など]
**UI生成ツール**: Google Stitch

| ディレクトリ | 役割 |
|---|---|
| `/src/components` | UIコンポーネント (Stitch生成コードの格納先) |
| `/src/app` | ページルーティング・メイン構成 |
| `/public` | アセット・画像類 |
| `/docs` | 外部仕様書・設計ドキュメント |

## HOW: 開発ワークフローとルール
**基本コマンド**:
- 開発起動: `npm run dev`
- ビルド: `npm run build`
- Lint: `npm run lint`

**Stitch連携ルール**:
1. Stitchで生成したUIコードはそのまま上書きせず、既存コンポーネントとの整合性を確認してマージすること。
2. デザインや実装の詳細は、必ず以下の外部ドキュメントを参照すること。

詳細は @path/to/import 構文で追加ファイルをインポート:
- デザインシステムとUI仕様 @DESIGN.md
- 要件定義 @PRD.md
